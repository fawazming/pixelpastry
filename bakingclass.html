Unlock the World of Web Design

Enroll in Pixel Pastry's Web Design Class!

Your Gateway to Digital Creativity!

Immerse yourself in the exciting world of web design and development.

In our web design class, you'll learn to craft stunning websites and user interfaces, and gain hands-on experience in coding and building websites that captivate users.

  • Learn from an experienced web design and development professional passionate about teaching
  • Learn to use popular web design tools and software to bring your creative visions to life.
  • Learn the art of creating websites that adapt seamlessly to different devices and screen sizes.
  • Get hands-on experience in coding and building websites.

10 Days at a Glance

01

Introduction to Web Development

- Understanding frontend development, HTTPS, DNS, and web hosting
- Exploring web browsers
- Introduction to Visual Studio Code
- Getting started with GitHub

02

More HTML and Git Knowledge

- HTML essentials: headings, paragraphs, lists, links, and images
- Expanding Git knowledge

03

Building Forms with HTML and Git Remote

- Creating various form elements: input types, textarea, select
- Enhancing Git skills: working with remote repositories

04

Advanced HTML and Project Class

- Introduction to external CSS for styling
- Exploring classless CSS libraries
- Designing a landing page
- Expanding Git knowledge

05

Project Showcase, Quiz, and Reward

- Participants showcase their completed projects
- Quiz on the course material
- Recognition and reward

06

Introduction to CSS

- Understanding the role of CSS
- Selectors and applying styles to HTML elements
- Styling text, colors, and backgrounds

07

Box Model and Layout

- Understanding the CSS box model
- Creating layouts with CSS
- Implementing responsive design principles

08

Advanced CSS Styling

- Working with CSS transitions and animations
- Adding custom fonts and icon libraries
- Styling forms and form elements

09

Project Development - Personal Portfolio

- Apply HTML and CSS skills to create a personal portfolio website
- Incorporate a navigation menu, showcase projects, and a contact form
- Ensure the website is responsive and optimized

10

Project Deployment

- Overview of web hosting and domain setup
- Deploying the personal portfolio website to a web server
- Final review and sharing the project with others

Find out everything you need to know and what you will be learning

The practical projects and hands-on exercises ensure that participants gain valuable experience in the exciting world of web design.
